## Class-identity-carrying logic per room

**Stillness = practice-led.** The room IS a practice. Breath-work, postural-anchor, non-doing. Mantra supplies the anchor sound (soham, Om). Wisdom banner is the occasional reminder ("Still the vṛttis" — YS 1.2). Principle and sankalp violate the pre-cognitive posture.

**Connection = mantra-led.** Bhakti is fundamentally japa / nāma-smaraṇa terrain. The room's medicine is the repeated name. Practice (anāhata breath, loving-kindness) supports. Wisdom banner occasional. Principle disrupts devotional state.

**Release = practice-led (somatic) + mantra (Shaiva).** The body stores what the mind cannot. Processing contraction cognitively re-contracts it. Shaiva mantras supply the "letting-go-is-sacred" field. Wisdom banner only in Ch 2.14 "endure like seasons" style — must stay sparse.

**Clarity = principle-led.** Discernment IS the surfacing of a principle — the L1/L2/L3 scaffold matches the user's moment exactly. Practice (single-decision, witness-drill, viveka exercise) supports. Wisdom teaching deepens. Sankalp before clarity is the Gita 2.63 trap — resolve-without-discrimination.

**Growth = principle + sankalp co-primary.** Growth is svadharma / abhyāsa / becoming. Principle supplies the scriptural substrate (already anchored: `gita_svadharma_is_what_you_have`). **Sankalp supplies the user-authored resolve — currently absent from pool.** Practice (abhyāsa drill) supports. Wisdom reflection post-runner.

**Joy = sankalp-led.** Already authored correctly with 3 subslots (gratitude/blessings/seva). Mantra supports with devotional anchors (Puṛṇamadaḥ, Kṛṣṇa, Lakṣmī). Practice is secondary (joy doesn't need procedure). Principle desiccates.
